# Виртуальные заметки для NVDA
Это дополнение (Virtual Notes for NVDA) создаёт виртуальные заметки, которые может читать программа экранного доступа NVDA. Заметки сохраняются между сеансами и автоматически записываются на диск.

Дополнительная информация, руководства и обновления:
* Официальный сайт: https://julianolopes.com.br/complemento-notas-virtuais-para-o-nvda
* Демонстрационное видео на YouTube: https://www.youtube.com/watch?v=yKJqS4imKHg

Доступные сочетания клавиш:  
* NVDA + Alt + A: добавить выделенный текст в память как новую заметку. Если текст не выделен, открывается диалог для ручного ввода, где можно использовать кнопки, чтобы:
  * Создать новую заметку.
  * Заменить текущую заметку.
  * Добавить введённый текст в начало текущей заметки.
  * Вставить введённый текст сразу после текущей строки активной заметки.
  * Добавить введённый текст в конец текущей заметки.
  * Сохранить изменения в текущей заметке (если редактируется текстовая заметка).
  * Переименовать текущую голосовую заметку (если активная заметка является аудиозаметкой).
* NVDA + Alt + V: добавить новую заметку напрямую из буфера обмена.
* NVDA + Shift + Alt + A: вставить выделенный текст как новую строку сразу после текущей строки активной заметки.
* NVDA + Alt + J: перейти к предыдущей заметке.
* NVDA + Alt + L: перейти к следующей заметке.
* NVDA + Shift + Alt + J: переместить текущую заметку на одну позицию влево (изменить порядок).
* NVDA + Shift + Alt + L: переместить текущую заметку на одну позицию вправо (изменить порядок).
* NVDA + Alt + U: прочитать или воспроизвести текущую заметку.
* NVDA + CONTROL + SHIFT + U: вставить текущую заметку в текущее приложение.
* NVDA + Alt + S: заменить заметку в текущей позиции (сначала нужно выделить текст).
* NVDA + Alt + D: удалить заметку в текущей позиции.
* NVDA + Alt + G: записать голосовую заметку (16 бит, 44,1 кГц, моно). Нажмите ещё раз, чтобы остановить запись.
* NVDA + Shift + Alt + U: приостановить или возобновить запись либо воспроизведение текущей голосовой заметки.

Если текущая заметка содержит несколько строк (например, текст скопирован из текстового редактора), можно использовать эти сочетания клавиш:  
* NVDA + Alt + I: перейти к предыдущей строке текущей заметки.
* NVDA + Alt + K: перейти к следующей строке текущей заметки.
* NVDA + Alt + O: прочитать текущую строку текущей заметки.
* NVDA + CONTROL + SHIFT + O: вставить текущую строку заметки в текущее приложение.

Примечание: расположение клавиш имитирует клавиши-стрелки (I = вверх, L = вправо, K = вниз, J = влево).  
После использования сочетания клавиш будет воспроизведён специальный звук, указывающий на выполненное действие.
