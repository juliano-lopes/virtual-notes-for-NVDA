Notas virtuais para NVDA

Este complemento (Virtual Notes for NVDA) cria notas virtuais que podem ser lidas pelo leitor de telas NVDA. Aqui estão as etapas disponíveis:
Selecionar texto; * Use atalhos (todos começam com as teclas “NVDA + Alt”). Ao pressionar essas teclas combinadas com letras específicas, você pode realizar as seguintes ações:
 A: Adicione uma nova nota; * J: Vai para a nota anterior; L: Vai para a próxima nota; * U: Leia a nota atual; * S: Substitui a nota na posição atual (você precisa selecionar algum texto primeiro); * D: Exclua a nota na posição atual. Se a nota atual contiver várias linhas (como texto copiado de um editor de texto), use estas letras adicionais:
 I: Move para a linha anterior da nota atual; * K: Move para a próxima linha da nota atual; * O: lê a linha atual da nota atual. Observação: As posições das teclas imitam as teclas de seta (I = para cima, L = para a direita, K = para baixo e J = para a esquerda).
Após usar um atalho, um som específico será reproduzido para indicar a ação realizada.
