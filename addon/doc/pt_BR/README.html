Notas virtuais para NVDA

Este complemento, Notas Virtuais para o NVDA (Virtual Notes for NVDA) adiciona textos à memória do computador que você pode utilizar como notas virtuais para serem lidas pelo leitor de telas NVDA. As notas são persistentes e salvas automaticamente no disco.

Para mais informações, tutoriais e novidades, acesse: * Site Oficial: https://julianolopes.com.br/complemento-notas-virtuais-para-o-nvda * Vídeo de demonstração no YouTube: https://www.youtube.com/watch?v=yKJqS4imKHg

Aqui estão os atalhos disponíveis:
* NVDA + Alt + A: Adicionar o texto selecionado como uma nova nota. Se nenhum texto estiver selecionado, abre uma caixa de diálogo para digitação manual, onde é possível usar os botões para: * Criar uma nova nota. * Substituir a nota atual. * Adicionar o texto digitado ao início da nota atual. * Inserir o texto digitado logo após a linha ativa da nota. * Adicionar o texto digitado ao final da nota atual. * Salvar alterações na nota atual (se for nota de texto). * Renomear a nota de voz atual (se a nota ativa for de áudio). * NVDA + Alt + V: Adicionar uma nova nota diretamente da área de transferência. * NVDA + Shift + Alt + A: Inserir o texto selecionado como uma nova linha logo após a linha ativa na nota atual. * NVDA + Alt + J: Ir para a nota anterior. * NVDA + Alt + L: Ir para a próxima nota. * NVDA + Shift + Alt + J: Mover a nota atual uma posição para a esquerda (reorganizar). * NVDA + Shift + Alt + L: Mover a nota atual uma posição para a direita (reorganizar). * NVDA + Alt + U: Anunciar ou reproduzir a nota atual. * NVDA + CONTROL + SHIFT + U: Colar a nota corrente no aplicativo atual. * NVDA + Alt + S: Substituir a nota na posição atual (é preciso selecionar algum texto primeiro). * NVDA + Alt + D: Excluir a nota da posição atual. * NVDA + Alt + G: Gravar uma nota de voz (16 bits 44.1 kHz Mono). Pressione novamente para parar a gravação. * NVDA + Shift + Alt + U: Pausar ou retomar a gravação ou reprodução da nota de voz atual.

Se a nota atual contiver várias linhas (como texto copiado de um editor de texto), use os seguintes atalhos: * NVDA + Alt + I: Mover para a linha anterior da nota atual. * NVDA + Alt + K: Mover para a próxima linha da nota atual. * NVDA + Alt + O: Anunciar a linha atual da nota atual. * NVDA + CONTROL + SHIFT + O: Colar a linha atual da nota no aplicativo atual.

Nota: As posições das teclas são similares às setas do teclado (I = para cima, L = para a direita, K = para baixo e J = para a esquerda).
Depois de usar um atalho, um som específico será reproduzido para indicar a ação realizada.
